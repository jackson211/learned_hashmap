# Learned HashMap
> HashMap that optimized with regression model.

## Installation
Requirement: cmake  

To build
```sh
cmake --build build
```

## Usage example

To run Learned HashMap:
```sh
./run.sh [DATA_FILE]
```
Example:
```sh
./run.sh test.txt
```

To run RTree:
```sh
cd rtree
./run_rtree.sh [DATA_FILE]
```
Example:
```sh
cd rtree
./run_rtree.sh ../test.txt
```
